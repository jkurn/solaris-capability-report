# Solaris AI Workforce Capability Report

A sample capability report demonstrating how organizations can be benchmarked on AI workforce readiness, derived from the ESI ThoughtLab "Driving ROI Through AI" study (n=1,200 firms).

**Live:** https://solaris.claudecode.studio

## What's inside

A single-page report scoring a fictional company ("Apex Financial Group") on:

- 4 core dimensions where Solaris solves directly (Talent Density, Workforce AI Fluency, Culture & Community, ROI Measurement)
- 6 supporting context dimensions
- A gap-to-feature mapping showing how Solaris closes each gap

## Stack

Static HTML + Tailwind (CDN) + inline SVG. No build step.

## Methodology

All benchmarks reference ESI ThoughtLab's "Driving ROI Through AI" report (Sep 2020). Demo company is fictional.
